# 절판서점
> 역경매로 절판된 도서를 최저가로 제공하는 서비스입니다.

역경매란 기존의 수요자 : 공급자 = n : 1 경매 방식과 달리, 수요자 : 공급자 = 1: n 형태로 공급자들의 가격 경쟁을 유도하는 리버스 옥션 방식입니다.
역경매를 통해 가장 좋은 조건을 제시한 판매자를 구매자가 선택하여 거래할 수 있는 사용자 중심 서비스입니다.
